# chromeapp
Chrome App for Miniurl.id

Install Procedure:
1. Download the .zip file
2. Extract the chromeapp-master.zip file
3. Open chrome://extensions
4. Turn on developer mode
5. Click Load Unpacked
6. Select chromeapp-master folder
